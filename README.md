##  Обзор проекта
Реализация алгоритмов машинного обучения **с нуля** с использованием набора данных California Housing. В проекте демонстрируются реализации градиентного спуска как для линейной, так и для логистической регрессии, а также сравнение и оценка моделей.

##  Информация о наборе данных

### Источник
- **Набор данных:** California Housing Dataset
- **Происхождение:** Перепись Калифорнии 1990 года
- **Источник:** Scikit-learn datasets
- **Ссылка:** https://scikit-learn.org/stable/modules/generated/sklearn.datasets.fetch_california_housing.html

### Признаки (фичи)
- **longitude, latitude:** Географические координаты
- **housing_median_age:** Медианный возраст домов
- **total_rooms:** Общее количество комнат
- **total_bedrooms:** Общее количество спален
- **population:** Население в районе
- **households:** Количество домохозяйств
- **median_income:** Медианный доход (масштабированный)
- **median_house_value:** Целевая переменная (медианная стоимость дома в $)
- **ocean_proximity:** Категориальный признак (NEAR BAY, <1H OCEAN, INLAND и т.д.)

### Статистика набора данных
- **Примеров:** 20 640
- **Признаков:** 9 (8 числовых + 1 категориальный)
- **Задача:** Регрессия (преобразована в классификацию для логистической регрессии)

##  Особенности проекта

###  Часть 1: Линейная регрессия с нуля
- Реализован алгоритм градиентного спуска
- Функция потерь MSE с ручным вычислением градиента
- Поддержка пакетного и мини-пакетного градиентного спуска
- Эксперименты со скоростью обучения и визуализация

###  Часть 2: Логистическая регрессия с нуля
- Реализация бинарной классификации
- Сигмоидная функция активации
- Логарифмическая функция потерь с L2-регуляризацией
- Ручное вычисление градиента

###  Часть 3: Сравнение моделей
- Пользовательская логистическая регрессия vs Дерево решений
- Метрики производительности: Точность, Precision, Recall, F1-Score
- Визуализация матрицы ошибок и ROC-кривой
